Multiplayer Browser

Browse any website together in real time. Shared navigation, scroll sync, chat, voice, and built-in proxy. Install the Chrome extension to start.

How to install

1

Download the extension

Go to the GitHub repo and clone or download the extension folder.

2

Open Chrome Extensions

In Chrome, go to chrome://extensions, turn on Developer mode (top right), then click Load unpacked.

3

Select the extension folder

Choose the extension folder you downloaded. The Multiplayer Browser icon will appear in your toolbar.

4

Start browsing together

Click the extension icon, create or join a room, and share the room ID with others. Everyone’s navigation and scroll stay in sync.
