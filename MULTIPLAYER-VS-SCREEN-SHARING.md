# Multiplayer Browser vs Screen Sharing

## TL;DR

**Screen sharing** = One person broadcasts their screen as video; everyone else watches.

**Multiplayer Browser** = Everyone browses the same page in their own browser. URL and scroll sync so you explore together. Everyone can interact.

---

## Comparison


|                      | Multiplayer Browser                       | Screen Sharing                       |
| -------------------- | ----------------------------------------- | ------------------------------------ |
| **What others see**  | Their own browser tab (same URL + scroll) | Video stream of one person's screen  |
| **Who can interact** | Everyone (click, scroll, type)            | Only the person sharing              |
| **Bandwidth**        | Low (URL, scroll, chat)                   | High (full video stream)             |
| **How it works**     | Syncs page state                          | Streams pixels                       |
| **Use case**         | Co-browsing, exploring together           | Presenting, support, watching a demo |


---

## Multiplayer Browser

- Each person has their **own** browser tab
- Everyone sees the **same URL** and **scroll position** synced in real time
- Everyone can **interact** with the page (click, scroll, type)
- Syncs only: URL, scroll position, chat **not** full video
- Works on **any website** no special permissions from the site
- **Low bandwidth** ideal for slow connections

## Screen Sharing

- One person **shares** their screen (or a window)
- Others see a **video stream** of that screen
- Only the **sharer** can interact; others just watch
- **High bandwidth** full video stream
- Often requires screen-share permission; some apps block it

---

## When to use which

**Use Multiplayer Browser when:**

- You want to browse a website together and everyone can click/scroll
- Bandwidth is limited
- You're exploring docs, shopping, or any web content as a group

**Use screen sharing when:**

- You're presenting a slide deck or demo
- You need to show something outside the browser (desktop, IDE, etc.)
- One person is driving and others are just watching
